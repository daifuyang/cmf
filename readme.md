# Gincmf1.0.0 让开发变得简单而快乐

## Gin1.0.0主要特性
* 强劲内核，基于Gin框架打造
* mvc框架支持
* 路由支持
* 多主题支持
* REST API支持
* 独立核心代码包cmf

## 环境推荐
> Go1.11+  

> nginx-1.17.10+ 

> mysql5.7+
